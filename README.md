# Retrogade

A craft-first social + portfolio platform focused on identity, aesthetics, and intentional creation.  
Built as a long-term, security-conscious system - not an engagement farm.

## Core Principles
- No algorithmic feed
- Security & abuse prevention designed in early
- Portfolio-first, not clout-first
- Anti-hustle UX
- Human-scale social features
